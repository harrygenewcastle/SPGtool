Interactive game

Let us consider now the following protocol:

Knowledge: A:[s, KAS, A, B, NA], B:[KBS, A, B], S:[A, B, E, KAS, KBS, KES, K] E:[A, B, NE, KES, fake]
Steps:

  1. A -> S: A, B, #NA
  2. S -> A: {#NA, #K, B, {K, A}KBS}KAS
  3. A -> B: {K, A}KBS, {#s}K

In this protocol, which a simplified version of the original Needham-Schroeder protocol, the server adds the explicit identity of the sender in the encrypted key for the recipient. Can you figure out a way for E to know the session secret? Can you figure out a way for B to know "fake"?
